Description of Files


1) NEW NERC TARIFFS
CSV files for all the new tariffs approved by the Nigerian Electricity Regulation Commission effective from 1st September 2020.

2) New Tariff Service Bands and Descriptions
Summary of the 5 different services bands in terms of duration of service (hours per day)

3) Old and New Tariff Class and Description
A link between the new tariff classes and old classes prior to September 1st 2020.
